Magic Film is a web platform that generates **personalized cinematic analysis using artificial intelligence**. Unlike IMDb or Letterboxd — which show user-written reviews with inconsistent quality — Magic Film produces its own structured analysis tailored to the character of each film.

The system automatically detects whether a film is **mass entertainment** or **auteur cinema**, and generates a completely different analysis for each case. All analysis is delivered in Spanish, organized in thematic layers, and cached in the database to avoid repeating work.

## What makes Magic Film different

<CardGroup cols={2}>
  <Card title="Adaptive AI analysis" icon="brain">
    Films are automatically classified as Entertainment or Deep/Arthouse. The AI generates a completely different analysis structure for each type.
  </Card>
  <Card title="Powered by Google Gemini" icon="sparkles">
    Uses Gemini 1.5 Flash to produce structured, layered cinematic analysis — not generic summaries.
  </Card>
  <Card title="Real film data via TMDB" icon="database">
    Film metadata, genres, keywords, posters, and ratings come directly from The Movie Database API v3.
  </Card>
  <Card title="Smart caching" icon="bolt">
    Generated analyses are stored in MySQL. Repeat requests are served instantly — no AI calls, no cost.
  </Card>
  <Card title="Role-based access" icon="shield">
    Three user tiers: Guest (browse), Registered (generate & save), and Admin (full management panel).
  </Card>
  <Card title="Favorites & history" icon="heart">
    Registered users can save favorite films and access their full search history.
  </Card>
</CardGroup>

## Analysis types

| Mode | Target films | Generated content |
|------|-------------|-------------------|
| **Entertainment** | Blockbusters, comedies, franchises | Spoiler-free summary · Epic moments · Easter eggs · Production trivia |
| **Deep** | Auteur films, dramas, folk horror | Visual symbolism · Themes · Cultural context · Director analysis |
