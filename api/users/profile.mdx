---
title: 'User profile'
description: 'Retrieve the profile of the currently authenticated user.'
api: 'GET /api/usuarios/perfil'
---

Returns the full profile record for the user identified by the JWT in the `Authorization` header. The `password_hash` field is always excluded from the response.

<Warning>
  Authentication is required. The request must include a valid JWT.
</Warning>

## Headers

<ParamField header="Authorization" type="string" required>
  `Bearer <token>` — JWT obtained from [POST /api/auth/login](/api/auth/login).
</ParamField>

## Response

<ResponseField name="usuario" type="object" required>
  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="number">
      Internal user ID.
    </ResponseField>
    <ResponseField name="nombre_usuario" type="string">
      Username.
    </ResponseField>
    <ResponseField name="email" type="string">
      Email address.
    </ResponseField>
    <ResponseField name="nombre_completo" type="string">
      Display name, or `null` if not set.
    </ResponseField>
    <ResponseField name="rol" type="string">
      Account role. One of `invitado`, `registrado`, or `admin`.
    </ResponseField>
    <ResponseField name="nivel_cinefilo" type="string">
      Cinephile level. One of `explorador`, `entusiasta`, `experto`, or `null`.
    </ResponseField>
    <ResponseField name="onboarding_completo" type="boolean">
      Whether the user has completed the onboarding flow.
    </ResponseField>
    <ResponseField name="activo" type="boolean">
      Whether the account is active.
    </ResponseField>
    <ResponseField name="creado_en" type="string">
      ISO 8601 timestamp of account creation.
    </ResponseField>
    <ResponseField name="actualizado_en" type="string">
      ISO 8601 timestamp of last update.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<RequestExample>

```bash cURL
curl --request GET \
  --url http://localhost:3000/api/usuarios/perfil \
  --header 'Authorization: Bearer <your-token>'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "usuario": {
    "id": 42,
    "nombre_usuario": "jdoe",
    "email": "jdoe@example.com",
    "nombre_completo": "Jane Doe",
    "rol": "registrado",
    "nivel_cinefilo": "entusiasta",
    "onboarding_completo": true,
    "activo": true,

{

```

</ResponseExample>
