Magic Film is live as an interactive frontend demo. You can explore the full user flow right now — search for a film, set your cinephile level, and view an AI analysis — directly in your browser.

<Note>
  Backend integrations (TMDB API, Google Gemini, and MySQL) are still in development. The current demo at [magic-film.vercel.app](https://magic-film.vercel.app) showcases the complete UI and navigation flow with mockup data.
</Note>

## Get started

<Steps>
  <Step title="Visit the live demo">
    Open [magic-film.vercel.app](https://magic-film.vercel.app) in your browser. You land on `index.html` — the home page — which introduces the platform and provides entry points to log in or register.
  </Step>

  <Step title="Create an account">
    Click **Registrarse** to open `registro.html`. Fill in your username, email, full name, and password to create a registered account.

    Registered accounts unlock analysis generation, saving favorites, and personalized recommendations. Guests can browse and view existing analyses without registering.
  </Step>

  <Step title="Complete onboarding">
    After registration you are directed to `onboarding.html`. Here you select your **cinephile level** — one of three tiers stored on your profile:

    | Level | `nivel_cinefilo` value | Description |
    |-------|------------------------|-------------|
    | Explorer | `explorador` | New to film analysis |
    | Enthusiast | `entusiasta` | Watches widely across genres |
    | Expert | `experto` | Deep knowledge of film history and theory |

    You also choose your preferred genres from the available list (Action, Drama, Comedy, Horror, Sci-Fi, Thriller, Romance, Psychological, Animation, Documentary, Historical, Musical). These preferences drive personalized recommendations once backend integrations are live.
  </Step>

  <Step title="Search for a film">
    You are taken to `busqueda.html` after onboarding. Type a film title into the search bar — for example, **Midsommar**. The backend queries the TMDB API to return matching results with posters, ratings, and release years.
  </Step>

  <Step title="View the AI analysis">
    Select a film from the results to open `analisis.html`. Magic Film automatically classifies the film as either **Entertainment** or **Deep** and displays the corresponding analysis layers.

    - Entertainment films show: spoiler-free summary, epic moments, easter eggs, and production trivia.
    - Deep/arthouse films show: narrative analysis, visual symbolism, cultural context, technical craft, and conclusion.

    See [Analysis modes](/features/analysis-modes) for a full breakdown of each layer.
  </Step>
</Steps>

## Pages in the application

The frontend is composed of the following HTML pages:

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page and entry point |
| Login | `login.html` | Returning user authentication |
| Register | `registro.html` | New account creation |
| Onboarding | `onboarding.html` | Cinephile level and genre preference setup |
| Search | `busqueda.html` | Film search powered by TMDB |
| Analysis | `analisis.html` | AI-generated analysis display |
| Admin panel | `admin.html` | Film and analysis management (admin role only) |

## User roles

| Role | What you can do |
|------|-----------------|
| **Guest** | Search for films and view existing analyses |
| **Registered** | Everything above, plus generate new analyses, save favorites, and receive personalized recommendations |
| **Admin** | Full film management, analysis regeneration, and monitoring panel |
